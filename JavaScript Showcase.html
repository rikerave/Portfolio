JavaScript Showcase

This section of my portfolio is dedicated to showcasing various JavaScript skills and projects. These examples are very simple yet dynamic and adds a great touch to any website or project.

Example 1: Events

Click the button to see how JavaScript responds to user actions.

In this example, a 'click' event listener is attached to the button element. When the button is clicked, the event listener triggers a function that updates the text content of a paragraph element to display "Button clicked!". This demonstrates the use of the addEventListener method to listen for user actions and respond with dynamic changes on the webpage, making it interactive and engaging. Through this method, JavaScript allows developers to create responsive, user-friendly web applications.

Example 2: Functions

This example demonstrates the power of JavaScript functions. Enter two numbers to calculate their sum.

In this example, a function named calculateSum is defined to take two arguments, add them together, and return the result. When the user inputs numbers and clicks the "Calculate Sum" button, an event listener triggers this function, passing the input values as arguments. The function's return value is then displayed to the user. This demonstrates not only the syntax for creating and invoking functions in JavaScript but also how functions can interact with webpage elements to create dynamic and interactive content.

Example 3: Loops

This example demonstrates the use of lops in JavaScrript. Click the button to sum the number in the array.

When the "Sum Numbers" button is clicked, a JavaScript function is triggered, which uses a 'for' loop to go through each element in the array. As it iterates, it adds each number's value to a running total. Once the loop has processed all array items, the resulting sum is displayed on the page. This not only showcases how to implement loops but also illustrates how they are a powerful tool for handling tasks involving collections of items, such as arrays.

Example 4: Conditionals

Enter a number to see message based on the value. This example shows how conditionals (if-else statements) can control the flow of a program.

In the conditional example, we use an if-else statement to execute different code based on user input. The user is prompted to enter a number, which is then evaluated by the JavaScript code to determine whether it is less than, equal to, or greater than 10. Conditionals are a fundamental part of programming, allowing for decision-making processes within code based on logical or comparative conditions.

When the user enters a value and clicks the "Show Message" button, the input is captured, and the code checks the value. If the input is not a number, it prompts the user to enter a valid number. If it is a number, further checks determine which message to display. The program uses a series of if-else statements to compare the input against the conditions defined in the code, showcasing how conditional logic can be used to control the flow of a program and provide interactive feedback to the user.

Example 5: Variables

Enter your name and click the button to see how JavaScript uses variables to store and display data.

In this example, a variable is used to store the user's name entered in the input field. When the user clicks the "Greet Me" button, an event listener triggers a function that retrieves the input value, stores it in a variable, and then uses this variable to construct a greeting message. This demonstrates the fundamental concept of variables in JavaScript, which are used to store data that can be manipulated and displayed back to the user, enhancing the interactivity of web pages.

Example 6: Working with Numbers

Enter a number and click the button to double its value. This example demonstrates basic arithmetic operations with numbers in JavaScript.

In this example, JavaScript's capability to perform arithmetic operations is showcased. When the user enters a number and clicks the "Double the Number" button, an event listener triggers a function that retrieves the input value, converts it into a numerical data type if necessary, and then applies an arithmetic operation to double the value. This value is then displayed back to the user, demonstrating how numbers are handled and manipulated in JavaScript to perform calculations and dynamically update web page content.

Example 7: Working with Arrays

Enter a word or phrase and click the button to add it to an array. Click another button to display all items in the array. This demonstrates how arrays can store multiple values in JavaScript.

In this example, an array is used to collect and store multiple user inputs. Each time the user enters a value and clicks the "Add to Array" button, the input is added to an array. Then, when the "Display Array" button is clicked, the content of the entire array is displayed to the user. This illustrates basic array operations in JavaScript, including creating arrays, adding items, and iterating over them to display their contents.
